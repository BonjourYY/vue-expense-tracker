### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/BonjourYY/vue-expense-tracker
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter Code in Command
   ```sh
   npm run dev
   ```
